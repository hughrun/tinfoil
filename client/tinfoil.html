Hello! This is a demo of the concepts discussed in this VALA conference paper in February 2016. It uses Meteor with Mylar, allowing lending libraries to control their assets and gain insights into lending behavour, whilst protecting library users' privacy by encrypting all loans data before it is sent to the server. This site is simply a demonstation of possible functionality - it's not an attempt to build a library management system.

You can find out more at www.hughrundle.net/tinfoil.
